# 1. 产品定位

面向美国用户的 AI personalized crystal bracelet 定制网站。用户可以通过意图风格推荐、手腕试戴推荐、东方出生能量推荐三种方式获得可购买的水晶手串方案，并在定制结果中直接完成下单。

英文主定位建议：Personalized Crystal Bracelets, Designed for Your Mood, Personality, Birth Energy, and Wrist.

核心体验不是让用户浏览大量 SKU，而是通过 AI 把用户的个性、手腕照片或出生能量信息转化成 3 套可购买的手串方案。

# 2. MVP 页面范围

MVP 网站包含 3 个页面。定制模式在“开始定制页”内完成切换，定制完成后进入“结果下单页”。下单不是弹窗，也不是外部独立窗口，而是在结果页中直接承接购买流程。

| 页面 | 页面目标 | 主要内容 | 关键 CTA |
| --- | --- | --- | --- |
| 首页 | 建立品牌信任，让用户理解网站卖什么。 | 店铺封面图、品牌介绍、核心卖点、样例手串、样例试戴效果、用户评价/信任元素。 | Start Customizing |
| 开始定制页 | 让用户选择并完成一种定制方式。 | 三个模式入口：意图风格推荐、手腕试戴推荐、东方出生能量推荐；每个模式有独立表单和引导。 | Generate My Bracelet |
| 结果下单页 | 展示定制结果，并直接完成购买转化。 | 3 套推荐方案、试戴图/分析摘要、价格、尺寸、材料、故事卡、收货与支付入口。 | Choose This Design / Buy Now |

# 3. 首页需求

首页先放店铺封面和品牌介绍，重点传达“这是一个能根据用户个性和能量定制水晶手串的网站”，而不是传统商品列表页。

| 模块 | 内容要求 | 交互要求 |
| --- | --- | --- |
| 店铺封面 | 首屏展示品牌氛围图或店铺封面，可包含模特佩戴、手串细节、自然光、水晶材质感。 | 首屏必须出现 Start Customizing 按钮。 |
| 品牌介绍 | 简短说明品牌、手工/定制属性、AI 个性化能力、美国用户友好的购买体验。 | 文案控制在 2-3 段内，避免玄学过重。 |
| 核心卖点 | 展示三类卖点：AI personalized design、virtual wrist try-on、birth energy inspired reading。 | 每个卖点可点击跳转到开始定制页对应模式。 |
| 样例结果 | 展示 2-3 张样例手串或试戴图，让用户预期最终输出长什么样。 | 样例图需标注 AI preview 或 sample。 |
| 信任元素 | 隐私说明、照片可删除、文化体验免责声明、支付/配送基础说明。 | 不进入复杂政策页，MVP 只展示简短提示。 |

# 4. 开始定制页总结构

开始定制页分为三个定制模式。用户进入页面后先看到三个入口卡片，也可以用 tabs/segmented control 切换模式。移动端三个入口纵向排列。

| 模式 | 页面名称建议 | 适合用户 | 系统输出 |
| --- | --- | --- | --- |
| 意图风格推荐 | Intention & Style Quiz | 不想上传照片，只想根据心情、目标、性格和审美快速获得推荐的用户。 | 3 套基于意图、风格、预算和可售库存生成的手串方案。 |
| 手腕试戴推荐 | Virtual Wrist Try-On | 想先看上手效果、确认颜色/尺寸/风格是否适合自己的用户。 | 3 套手串方案 + 佩戴在用户手腕照片上的 AI preview 图。 |
| 东方出生能量推荐 | Birth Energy Reading | 对东方文化、出生时间、五行能量感兴趣，希望获得更有故事感推荐的用户。 | 换算北京时间后生成八字/五行摘要，并输出 3 套文化灵感型手串方案。 |

三种模式都必须最终输出可购买方案，不允许只输出分析报告。每种模式默认推荐 3 套方案，最多 4 套。

# 5. 模式一：意图风格推荐

入口文案建议：Find a crystal bracelet that matches your personality, mood, and intention.

| 字段 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| 当前心情 | 单选/多选 | 必填 | calm、anxious、focused、romantic、energetic、healing 等。 |
| 想增强的方向 | 多选 | 必填 | confidence、love、focus、protection、balance、creativity 等。 |
| MBTI | 下拉/可跳过 | 选填 | 不知道可跳过，不能阻塞流程。 |
| 性格关键词 | 标签/文本 | 选填 | 用户可输入 1-5 个词。 |
| 穿搭风格 | 多选 | 选填 | minimalist、boho、elegant、dark、colorful、daily wear 等。 |
| 颜色偏好 | 多选 | 选填 | 喜欢/不喜欢的颜色都可填写。 |
| 预算区间 | 单选 | 必填 | 用于约束推荐方案价格。 |
| 手围 | 数字输入 | 选填 | 如果填写，结果页优先使用该尺寸。 |

推荐逻辑要求：

1. 根据用户的情绪、目标、风格、预算筛选可售 SKU 或可生产组合。
2. 根据 MBTI/性格关键词补充推荐理由，但不得把 MBTI 描述成科学诊断。
3. 每套方案必须包含方案名、主石、副石、颜色风格、珠径、尺寸建议、推荐理由、适合场景、价格、库存状态和下单按钮。

# 6. 模式二：手腕试戴推荐

该模式的核心价值是让用户看到“这串手串戴在我手腕上是什么效果”。它不是单纯上传图片，而是支持“当场拍照”和“本地上传”两条路径。

| 环节 | 需求 |
| --- | --- |
| 入口文案 | See your personalized bracelet on your own wrist. |
| 输入组合 | 性格/MBTI + 手腕照片 + 可选手围。 |
| 拍照方式 | 支持手机摄像头、电脑摄像头；前端调用浏览器 MediaDevices/getUserMedia。 |
| 上传方式 | 支持手机相册上传、桌面文件上传；接受 jpg、png、webp。 |
| 授权规则 | 必须由用户主动点击 Take Photo 后才请求摄像头权限，不得进入页面自动拉起摄像头。 |
| 拒绝授权 | 用户拒绝摄像头权限后，页面保留 Upload Photo 入口，流程不能中断。 |
| 拍摄引导 | 提示手腕平放、自然光、距离 20-40cm、露出完整手腕、避免遮挡、避免强滤镜。 |
| 图片质检 | 检测是否清晰、是否包含完整手腕、亮度是否足够、是否有严重遮挡。 |
| 失败处理 | 照片不合格、生成失败、接口超时均不扣生成次数，并提示重新拍摄或上传。 |
| 生成结果 | 生成手串自然佩戴在用户手腕上的 AI preview 图，同时展示 3 套可购买方案。 |
| 水印规则 | 免费试戴图必须加品牌水印和 AI preview 说明；购买后可提供无水印版本或打印故事卡。 |
| 隐私说明 | 照片仅用于生成试戴预览，可在账户中删除；不得用于广告训练或公开展示，除非用户明确授权。 |

试戴生成逻辑：

1. 先根据性格/MBTI/意图生成候选手串方案。
2. 再根据手腕图估算角度、手腕区域、光照方向和可视比例。
3. 将候选手串以合适珠径、透视、遮挡和光影贴合到手腕上。
4. 如果用户填写手围，尺寸建议优先使用手围；未填写时只做视觉估算，不承诺精确尺寸。

# 7. 模式三：东方出生能量推荐

该页面参考“根据生辰八字定制”的原始方向，但在美国市场中定位为 Eastern Birth Energy Reading / Chinese Five Elements inspired personalization。表达上要强调文化体验、个人风格和饰品建议，不做改命、发财、治疗等承诺。

| 字段 | 类型 | 是否必填 | 要求 |
| --- | --- | --- | --- |
| 出生日期 | 日期选择 | 必填 | MVP 支持公历日期；如果后续支持农历，需要单独切换入口。 |
| 出生时间 | 时间选择/时辰选择 | 必填 | 支持具体时间，也可选择子丑寅卯等传统时辰。 |
| 不知道出生时间 | 选项 | 可选 | 允许选择 Unknown，输出时标记分析精度降低。 |
| 出生地 | 城市搜索 | 必填 | 至少包含国家、州/省、城市，用于识别时区。 |
| 性别表达/佩戴偏好 | 单选/可跳过 | 选填 | 只用于风格与尺寸偏好，不用于做价值判断。 |
| 风格偏好 | 多选 | 选填 | daily、minimal、luxury、boho、gift 等。 |
| 预算区间 | 单选 | 必填 | 约束推荐方案。 |

北京时间换算与八字分析流程：

1. 用户提交出生日期、出生时间、出生地。
2. 系统根据出生地识别当地时区，并处理夏令时。
3. 将用户出生时间换算为北京时间点。
4. 使用北京时间进入八字排盘逻辑，计算年柱、月柱、日柱、时柱。
5. 统计金、木、水、火、土五行分布，识别偏强、偏弱、缺失或需要平衡的元素。
6. 将五行结果映射为颜色、材质、水晶方向和佩戴建议。
7. 输出 3 套可购买手串方案：平衡型、增强型、日常佩戴型。

五行与推荐映射要求：

| 五行 | 可用表达 | 可关联方向 |
| --- | --- | --- |
| 金 | clarity、structure、focus | 白水晶、月光石、银色配件、浅色系。 |
| 木 | growth、renewal、creativity | 绿幽灵、东陵玉、绿色系。 |
| 水 | calm、intuition、flow | 海蓝宝、黑曜石、蓝色/黑色系。 |
| 火 | energy、confidence、passion | 红玛瑙、石榴石、红色/紫红系。 |
| 土 | grounding、stability、protection | 黄水晶、虎眼石、茶晶、黄色/棕色系。 |

合规要求：

1. 页面必须展示免责声明：This reading is for cultural inspiration and jewelry personalization only.
2. 不得承诺改命、发财、治愈疾病、改变关系结果或预测未来。
3. 出生信息属于敏感个人信息，只展示必要摘要，并支持删除。

# 8. 结果下单页

定制完成后进入结果下单页。结果页既是定制结果展示页，也是购买页。不得弹出独立下单窗口。

| 区域 | 内容 | 交互 |
| --- | --- | --- |
| 推荐方案区 | 默认 3 套方案：方案名、主图/试戴图、主石、副石、颜色、珠径、价格、库存状态。 | 用户选择其中一套作为购买方案。 |
| 定制解释区 | 展示为什么推荐这套：个性理由、试戴效果或八字五行摘要。 | 可折叠查看详细分析。 |
| 尺寸确认区 | 手围、珠径、松紧偏好、是否作为礼物。 | 未填写手围时要求补充或选择标准尺寸。 |
| 商品配置区 | 数量、配件、包装、故事卡、是否保留试戴打印图。 | 选择后实时更新价格。 |
| 购买区 | 收货信息、支付入口、加入购物车/立即购买。 | 页面内完成，不打开独立下单弹窗。 |
| 再生成区 | 重新生成、切换模式、返回修改输入。 | 重新生成按次数规则处理。 |

订单生成规则：

1. 用户点击 Buy Now 后生成订单草稿，并锁定当前定制方案快照。
2. 支付成功后锁定库存，生成制作单。
3. 制作单需要包含定制来源、用户选择、手围、珠径、材料、故事卡文案和试戴图文件。
4. 如果推荐 SKU 缺货或下架，结果页必须提示替换方案，不能继续支付。
